# BumpOrBlast - 게임 디자인 문서

## 게임 개요
- **장르**: 2D 탑다운 슈터 (Top-down Shooter)
- **플랫폼**: PC (Unity 2D)
- **플레이 타임**: 약 5분
- **게임 목표**: 최대한 오래 생존하며 높은 점수를 획득

## 핵심 게임플레이

### 조작 시스템
- **이동**: 방향키 (↑↓←→)
- **공격**: 마우스 클릭 (마우스 방향으로 발사)
- **재장전**: R 키 (TODO)
- **회피**: 스페이스 바 (TODO)

### 사격 시스템
- 마우스 클릭으로 탄환 발사
- 탄창 용량 제한 (기본값: 10발)
- 탄창 UI에 잔탄수 표시
- 탄창 소진 시 재장전 필요
- 재장전 중에는 사격 불가

### 체력 시스템
- 플레이어 초기 체력: 100
- 적과 충돌 시 데미지 입음
- 체력이 0이 되면 게임 오버
- UI에 체력바 표시

## 적 시스템

### 적 AI
- 플레이어를 향해 직선 이동 (물리 엔진 사용)
- 플레이어와 충돌 시 데미지 입힘
- 체력 소진 시 사망

### 적 종류 (3종)
1. **일반 적** (Normal)
   - 기본 이동속도
   - 보통 체력
   - 보통 데미지

2. **빠른 적** (Fast)
   - 빠른 이동속도
   - 낮은 체력
   - 낮은 데미지

3. **강한 적** (Tank)
   - 느린 이동속도
   - 높은 체력
   - 높은 데미지

### 적 스폰 시스템
- 일정 시간 간격으로 자동 스폰
- 플레이어로부터 일정 거리 밖 랜덤 위치에 스폰
- 스폰 간격이 시간에 따라 점점 짧아짐 (난이도 증가)
- 최소 스폰 간격 제한 존재
- 3종류 적 중 무작위 선택하여 스폰
- 플레이어로부터 너무 멀어진 적은 자동 삭제 (성능 최적화)

## 점수 & 레벨업 시스템

### 점수 획득
- 적 격파 시 점수 획득
- 적 종류별로 다른 점수 부여

### 레벨업
- 일정 점수 달성 시 레벨업
- 레벨업 시 게임 일시정지
- 4가지 강화 옵션 중 1개 선택:
  1. **체력 증가** - 최대 체력 +20, 현재 체력 회복
  2. **이동속도 증가** - 이동속도 +10%
  3. **공격력 증가** - 탄환 데미지 +20%
  4. **탄창 크기 증가** - 탄창 용량 +5발

## 아이템 시스템 (Phase 5)

### 아이템 드롭
- 적 처치 시 일정 확률로 아이템 드롭
- 아이템은 바닥에 일정 시간 동안 유지
- 플레이어가 접촉하면 자동 획득
- 획득 시 효과음 및 시각적 피드백

### 아이템 종류
1. **경험치 블록** (XP Orb)
   - 획득 시 즉시 경험치 획득
   - 레벨업에 필요한 점수 일부 제공
   - 드롭 확률: 보통

2. **체력 회복** (Health Pack)
   - 획득 시 체력 회복 (20~30 HP)
   - 최대 체력 이상으로 회복되지 않음
   - 드롭 확률: 낮음

3. **폭탄** (Bomb)
   - 획득 시 화면 내 모든 적 즉시 제거
   - 화면 전체 폭발 효과 및 사운드
   - 처치한 적에 대해서도 점수 획득
   - 드롭 확률: 매우 낮음

4. **기타 파워업**[TODO]
   - 일시적 무적 상태
   - 일시적 공격력/연사력 증가
   - 일시적 이동속도 증가
   - 자석 효과 (아이템 자동 흡수)

## 리더보드 시스템 (Phase 5)
- 게임 종료 시 최고 점수 기록
- 상위 5개 기록 저장
- PlayerPrefs 또는 JSON 파일로 로컬 저장
- 게임 오버 화면에서 순위 표시
- 기록 정보:
  - 순위
  - 점수
  - 이름(게임 오버 화면에서 입력)

## 시각/사운드 연출

### 화면 효과
- **화면 흔들림**: 플레이어 피격 시
- **플로팅 텍스트**:
  - 재장전 시도 시 "Reloading!" 표시
  - 탄창 소진 상태에서 사격 시도 시 "No Ammo!" 표시
  - 레벨업 시 "LEVEL UP!" 표시
- **탄피 연출**: 사격 시 탄피 오브젝트가 물리 엔진으로 튕겨나감

### 파티클 효과 (선택사항)
- 적 사망 시 파티클
- 총구 섬광
- 피격 효과

## UI 구성

### 게임 플레이 UI
- **체력바**: 화면 상단 또는 좌측 상단
- **점수**: 화면 상단
- **레벨**: 점수 옆
- **탄창 UI**: 화면 하단 (총알 아이콘으로 잔탄수 표시)
- **조작법 안내**: 화면 구석에 배치
  ```
  이동: 방향키
  공격: 마우스 클릭
  재장전: R
  회피: 스페이스바
  ```

### 메뉴 UI
- **시작 화면**: 게임 시작 버튼
- **게임 오버 화면**: 최종 점수, 재시작 버튼
- **레벨업 화면**: 강화 선택 UI

### 고정 비율 UI (Phase 5)
- **다양한 화면 비율 지원**:
  - 기본 16:9 비율 대응
