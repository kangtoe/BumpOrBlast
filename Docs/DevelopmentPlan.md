# 개발 우선순위

## Phase 1: 핵심 메커니즘
> 게임의 기본 플레이가 가능한 최소 기능

1. 플레이어 이동 (방향키)
2. 마우스 방향으로 사격
3. 적 스폰 및 AI (플레이어 추적)
4. 충돌 감지 (탄환-적, 플레이어-적)
5. 체력 시스템

**목표**: 게임 루프의 기본 형태 완성

---

## Phase 2: 게임 루프
> 게임을 반복해서 플레이할 수 있는 구조

1. 점수 시스템
2. 레벨업 시스템
3. 게임 오버 및 재시작
4. 스폰 난이도 증가

**목표**: 완전한 게임 사이클 구현

---

## Phase 3: 폴리싱
> 게임의 완성도를 높이는 시각적/UX 요소

1. UI 구현 (체력바, 점수, 탄창)
2. 화면 흔들림
3. 플로팅 텍스트
4. 탄피 연출
5. 조작법 안내

**목표**: 플레이 경험 향상

---

## Phase 4: 추가 개발 계획
> 향후 구현 예정 기능

1. 발사 방식 변경 (반자동 → 자동)
2. 연사력 업그레이드 추가
3. 피격/파괴 사운드 효과 추가
4. 게임 시작 시 파티클 효과 추가

**목표**: 게임 피드백 및 전투 시스템 강화
