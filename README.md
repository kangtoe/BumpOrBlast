# JungleKrafton

2D 탑다운 슈터 게임 프로젝트

## 게임 소개
플레이어는 끊임없이 몰려오는 적들을 물리치며 최대한 오래 생존해야 합니다.
시간이 지날수록 난이도가 증가하며, 레벨업을 통해 캐릭터를 강화할 수 있습니다.

- **장르**: 2D Top-down Shooter
- **플랫폼**: PC (Unity)
- **플레이 타임**: 약 5분

## 주요 기능
- 방향키 이동 + 마우스 사격
- 3종류의 적 (일반/빠른/강한)
- 점수 & 레벨업 시스템
- 화면 효과 및 UI 연출
